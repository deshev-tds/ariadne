# Offsec: Prefer Official Docs Before Broad Web Search for Exact Syntax

Applies When:
- the corpus gives the method but exact flags are blocking
- version-specific behavior or config details matter
- tool syntax failures repeat after a method-level consult

Prefer:
- check official or project docs first
- use GitHub or project-maintained documentation for exact flags and config shape
- keep broad web search as a later fallback

Avoid:
- going straight to broad web search for syntax or flag details
- re-consulting the corpus repeatedly for version-specific exactness

Signal:
- syntax or flag failure
- version-specific blocker after methodology is already known

Status:
- promoted
